`OPENAI_API_KEY`

<br>

## Run Locally 🏃🏻

Clone the project

```bash
  git clone https://github.com/arjun-ms/SQL-Ease.git
```

Go to the project directory

```bash
  cd SQL-Ease
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
